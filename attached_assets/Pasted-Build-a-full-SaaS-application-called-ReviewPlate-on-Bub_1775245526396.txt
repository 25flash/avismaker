Build a full SaaS application called "ReviewPlate" on Bubble.io.
This is a smart NFC/QR review card platform that redirects customers
to review pages (Google, Airbnb, TripAdvisor, Trustpilot, etc.).
Follow ALL instructions below precisely using Bubble's native tools.

--- PAGE: activate ---

⚠️ IMPORTANT: Card activation is done ONLY by manually typing
the activation code printed on the physical card.
There is NO NFC scan, NO QR code scan, NO camera access
on this page. The user types the code themselves.

Use a multi-step group showing one step at a time.

STEP 1 — Manual code entry:
  Title: "Enter your card activation code"
  Subtitle: "The code is printed on the back of your ReviewPlate card"
  Input: large centered text input
    - Placeholder: "Ex: AB12CD"
    - Auto-uppercase: yes (add workflow: on input change →
      set input value = Input's value:uppercase)
    - Max characters: 6
  Button (Primary, amber): "Activate this card"

  Workflow on button click:
    1. IF input is empty → show inline error "Please enter your code"
    2. Search for Card where code = Input's value (uppercase)
    3. IF no Card found → show error message:
         "This code does not exist. Check the code on your card."
    4. IF Card found AND status = "disabled" →
         show error: "This card has been deactivated. Contact support."
    5. IF Card found AND status = "active" →
         show message: "This card is already activated."
         show button: "Go to the review page →"
         on click: open /c?code=[card code] in new tab
    6. IF Card found AND status = "inactive" →
         Store card in Custom State "current_card"
         → show Step 2

STEP 2 — Platform selection:
  Title: "What platform do you want to link?"
  Show 6 platform buttons in a 2-column grid, each with:
    - Platform icon (use image elements with logos)
    - Platform name
    - Short description (ex: "Redirect to your Google review page")

  Platforms:
    [ Google ]      [ Airbnb ]
    [ TripAdvisor ] [ Trustpilot ]
    [ Multi-link ]  [ Social media ]

  Each button: background white, border 1.5px #E5E7EB,
  border-radius 12px, on hover border #F59E0B
  Selected state: border 2px #F59E0B, background #FFFBEB

  On platform click:
    Store in Custom State "selected_platform"
    → show Step 3

STEP 3 — Platform configuration:
  Show different input group based on selected_platform:

  IF google:
    → Google Places Autocomplete input
      Label: "Search for your business on Google"
      Type: "establishment"
      On selection: store place_id + build review URL:
        https://search.google.com/local/writereview?placeid=[place_id]
      Store in Custom State "target_url"

  IF airbnb:
    → Form with file uploads and text fields:
      - Page title (text input)
      - Cover image (image file uploader — Bubble native)
      - House rules (multiline text area)
      - Check-in information (multiline text area)
      - WiFi network name (text input)
      - WiFi password (text input)
      - Local recommendations (multiline text area)
    Store all in Custom State "airbnb_data"

  IF tripadvisor OR trustpilot:
    → Text input: "Paste your review page URL"
    → URL validation: must start with https://
    Store in Custom State "target_url"

  IF multilink:
    → Logo upload (image file uploader — Bubble native)
    → Title (text input)
    → Subtitle (text input)
    → Background color picker (hex input or color picker plugin)
    → Accent color picker
    → Repeating group of link inputs (add up to 8):
        Each row: icon emoji input + label + URL
        "Add a link" button adds a new row
    Store in Custom State "multilink_data"

  IF social:
    → Profile photo (image file uploader — Bubble native)
    → Bio (short text input, max 120 chars)
    → Optional URL inputs for each platform:
        Instagram / TikTok / Facebook / YouTube / X (Twitter)
    Store in Custom State "social_data"

  Button "Continue →" → show Step 4

STEP 4 — Account & Business Profile:
  IF Current User is NOT logged in:
    Show signup form:
      - Email input
      - Password input
      - Confirm password input
      - "Create my account" button (amber)
    Below form: "Already have an account?" link → show login form
    On successful signup: log user in → continue to Step 5

  IF Current User IS logged in:
    Skip directly to Step 5

STEP 5 — Assign to a Business Profile:
  ⚠️ This step is always shown when user is logged in.
  It allows the user to associate the card with the right business.

  Title: "Which business do you want to link this card to?"

  CASE A — User has NO business profiles yet:
    Show a mini business creation form:
      - Business name (text input)
      - Business category (dropdown: Restaurant, Hotel, Bar, Shop,
        Beauty, Healthcare, Other)
      - Address (text input)
      - Logo (image file uploader — Bubble native)
      ⚠️ FREE PLAN LIMIT CHECK:
        Count of BusinessProfiles where owner = Current User
        IF count >= 1 AND Current User's plan = "free":
          Hide the creation form
          Show message:
            "Your Free plan is limited to 1 business profile."
            "Upgrade to Premium to add more businesses."
          Show "Upgrade to Premium →" button (amber)
          → on click: redirect to /billing
          STOP — do not allow card activation until user upgrades

  CASE B — User has 1 existing business profile:
    Show the profile card with name + logo
    Text: "This card will be linked to [Business Name]"
    Button: "Confirm and activate →"

  CASE C — User has 2 or more business profiles:
    Title: "Select the business to link this card to:"
    Show a selectable list of all their BusinessProfiles:
      Each item shows: logo thumbnail + business name + category
      Style: card with border, selected state = amber border
      User MUST select one before continuing
    Add a "+ Create a new business" link at the bottom
      (only if plan allows it — Premium: max 3, Pro: max 10,
       Business: unlimited — check count before showing)

  Store selected BusinessProfile in Custom State
  "target_business_profile"
  Button "Activate my card →" → show Step 6

STEP 6 — Activation + Success:
  Run these workflows in sequence:

  1. IF new business profile was created in Step 5:
       Create BusinessProfile:
         owner = Current User,
         name = form input,
         category = dropdown value,
         address = address input,
         logo = uploaded image file (Bubble file uploader value)

  2. Modify Card:
       status = "active"
       owner = Current User
       platform = Custom State "selected_platform"
       target_url = Custom State "target_url"
       business_profile = Custom State "target_business_profile"
       activated_date = Current date/time

  3. IF platform = "airbnb":
       Create AirbnbPage with all airbnb_data fields,
       set cover_image = uploaded file (from file uploader),
       link to this Card

  4. IF platform = "multilink":
       Create MultilinkPage,
       set logo = uploaded file,
       Create LinkItems for each link row,
       link to this Card

  5. IF platform = "social":
       Create SocialPage,
       set profile_photo = uploaded file,
       link to this Card

  6. Modify Current User → add Card to user's cards list

  7. Show success screen:
       Large checkmark icon (green #10B981)
       Title: "Your card is activated!"
       Subtitle: "It's now linked to [Business Name]"
       Two buttons:
         "Test my card" → open /c?code=[card code] in new tab
         "Go to my dashboard" → navigate to /dashboard
